Etapa 2 | Pesquisa e Ordenação

Na segunda etapa, exercitaremos os métodos de Pesquisa e Ordenação, utilizando os métodos estudados para realizar uma ordenação de uma classe criada por você. Para este desafio, é imprescindível o entendimento da função da estabilidade de algoritmos de ordenação.

Passos a Seguir:
  • Crie uma classe Candidato:
    • Possui um atributo nome (nome completo do candidato)
    • Um atributo partido (String)
    • Um inteiro chamado intencoesVotos, representando a quantidade de intenções de voto obtidas na pesquisa
    • Crie os métodos de acesso dos atributos (getters e setters)
    • Crie um construtor que recebe informações para inicializar todos os atributos
    • Crie também o método toString() para formatar a saída de forma organizada
  • Crie uma classe chamada PrincipalCandidatos:
    • Contém o método main
    • No main, crie um array de Candidato de tamanho aleatório entre 1 e 100
    • Preencha este array com objetos do tipo Candidato, com informações geradas aleatoriamente
    • É importante que possa haver candidatos com o mesmo nome e diferentes candidatos do mesmo partido
  • Classe OrdenarCandidatos:
    • Implemente métodos para ordenar os candidatos:
      • ordenaCandidatosPorNome
      • ordenaCandidatosPorVotos
      • ordenaCandidatosPorPartido
    • Esses métodos devem ordenar um array de Candidato baseado em diferentes critérios
  • Pesquisa Binária:
    • Na classe OrdenarCandidatos, crie um método estático chamado pesquisaBinariaCandidatos
    • Esse método deve pesquisar um Candidato pelo nome no array já ordenado
  • Teste do Método de Pesquisa:
    • Teste o método com o array ordenado pelo seu método de ordenação
    • Imprima as informações do Candidato que possui o nome informado pelo usuário, usando o método pesquisaBinariaCandidatos
